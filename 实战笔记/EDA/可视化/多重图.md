# 多重图绘制

## 绘制多张图

```python
g = sns.FacetGrid(df_train, col="type", col_wrap=2, height=4, aspect=1.5)
g.map(sns.violinplot, "dist");
plt.subplots_adjust(top=0.9)
g.fig.suptitle('Violin Distribution of Dist by Each Type', fontsize=25)
```
